QuickBookingModal 현재 구현 상태

2025-07-09 기준

📐 모달 기본 구조

  • 크기: max-w-7xl (1280px), height: 95vh
  • 배경색: #FBF9F5 (아이보리)
  • 애니메이션: fade-in, slide-in 효과
  • 스케일: 0.8배 축소 (transform scale-[0.8])
  • Portal 렌더링: document.body에 렌더링

🎨 헤더 디자인

// 헤더 구조 - 배경색: #016B9F (Entrip Primary Blue) - 제목: "예약 입력" / "예약 수정" (동적) - 부제목: "새로운 팀 예약을 작성하세요" - 버튼 그룹: absolute positioning, right: 30px - 취소 버튼 - 출력 버튼 (프린터 아이콘) - 저장 버튼

📋 3열 레이아웃 구조

Column 1 - 기본 정보

  • 팀 정보 (타입, 팀명)
  • 일정 (출발일, 도착일)
  • 구간 (출발지, 목적지)

Column 2 - 상품 구성

  • 항공편 (Repeater)
  • 차량 (Repeater)
  • 호텔 (Repeater)

Column 3 - 추가 정보

  • 팀 구성원 (인원수, 담당자)
  • 고객 정보 (대표자, 연락처, 이메일)
  • 정산 정보 (Repeater)
  • 추가 메모

열 간격: 8px (gap-x-2)

🔧 구현된 컴포넌트

입력 필드 컴포넌트

  • InputText (shadow-inner)
  • InputSelect (드롭다운)
  • InputDate (날짜 선택)
  • InputTime (시간 선택)
  • InputNumber (숫자 입력)
  • TextArea (여러 줄 텍스트)

레이아웃 컴포넌트

  • FieldBase (라벨, 에러 메시지)
  • Section (카드 스타일 그룹)
  • FormCard (내부 카드)
  • Repeater (동적 추가/삭제)
  • RepeaterItem (개별 항목)

기능적 특징

  • React Hook Form 통합
  • Zod 스키마 검증
  • ESC 키로 닫기
  • 외부 클릭으로 닫기
  • Focus Trap
  • Body overflow 관리

✅ 완료된 스타일 개선사항

  • 완료 열 간격 8px 고정 (gap-x-2)
  • 완료 헤더 동적 포지셔닝 (absolute positioning)
  • 완료 버튼 그룹 우측 30px 고정
  • 완료 입력 필드 shadow-inner 효과
  • 완료 추가 버튼 아이보리 배경 (#FBF9F5)
  • 완료 항공편 레이어 여백 축소 (p-4 → p-2.5)
  • 완료 모달 스케일 0.8배 적용

⚠️ 확인 필요 사항

  • 애니메이션 동작
  • 반응형 디자인 동작
  • 스크롤 동작
